# Why You Should Love Dr. Santos

## Overview

A fan tribute webpage defending Dr. Trinity Santos, a misunderstood character from the HBO Max medical drama *The Pitt*. This page presents a thoughtful case for why viewers should give the character a chance, analyzing her personality, actions, and backstory from the show.

## Features

- Responsive image layout with CSS floats
- Floating quote section with serif typography
- Image gallery with styled borders and shadows
- Organized content with numbered list and multi-column layout
- Clean, readable typography with justified text

## Technologies Used

- **HTML5** - Semantic page structure
- **CSS3** - Custom styling including:
  - Float-based layouts
  - Image containers with borders and shadows
  - Responsive max-width container
  - Custom typography (sans-serif body, serif for quotes)

## Project Structure
dr-santos-fanpage/
├── index.html # Main webpage content
├── style.css # All styling rules
└── README.md # This file


## Running Locally

1. **Save both files** in the same folder:
   - `index.html` (copy your HTML code)
   - `style.css` (copy your CSS code)

2. **Open the webpage**:
   - Double-click `index.html`, OR
   - Use Live Server in VS Code, OR
   - Drag the file into any web browser
